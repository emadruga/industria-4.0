| *Bloco/Pilar:* |  |

| *Resumo Descritivo* |
| :---- |
| A Colaboração Inter e Intraempresarial refere-se ao processo de trabalhar em conjunto, por meio de equipes multifuncionais e com parceiros externos, para alcançar uma visão e propósito compartilhados. A comunicação aberta estabelece canais formais e transparentes que permitem o compartilhamento efetivo de informações, ideias e conhecimento entre diferentes áreas, níveis hierárquicos e organizações, promovendo a cooperação, coordenação e integração necessárias para a transformação digital. |

| *Capacidades Relacionadas:* | Estilo de liderança democrático Cultura de confiança Compartilhamento de conhecimento Transparência organizacional Trabalho em equipe multifuncional |
| :---- | :---- |

# **Questão 1 - Comunicação aberta**

| Como a comunicação e o compartilhamento de informações acontecem entre diferentes equipes, áreas e parceiros externos na sua organização? |  |
| :---- | :---- |
| ***Capacidade em medição:*** | Comunicação aberta |

| *Respostas* |  |
| :---: | :---: |
| ***Peso*** | ***Item de resposta*** |
| 0 | A comunicação acontece de forma informal e ad hoc. Equipes trabalham em silos, e o compartilhamento de informações ocorre apenas casualmente, sem estrutura ou registro. |
| 1 | Canais formais são estabelecidos para comunicação e compartilhamento de informações entre equipes. Existem procedimentos definidos, mas a comunicação ainda é limitada e reativa. |
| 2 | Canais formais permitem que equipes cooperem em tarefas e projetos pontuais. Há instrumentos visíveis para compartilhamento de informações, mas a colaboração depende de mediação da liderança. |
| 3 | Equipes têm autonomia para fazer ajustes que facilitam a cooperação. A comunicação é mais proativa, com reuniões regulares e compartilhamento estruturado de informações entre áreas. |
| 4 | Equipes compartilham recursos, responsabilidades e riscos em projetos discretos e de longo prazo. A comunicação é bidirecional, transparente e baseada em dados quantificáveis. |
| 5 | Canais formais permitem a formação dinâmica de equipes para trabalhar em projetos multifuncionais com metas, recursos e KPIs compartilhados. A comunicação é totalmente transparente e integrada. |
| 6 | A organização opera como uma rede altamente adaptável com comunicação em tempo real. Equipes se formam e se ajustam dinamicamente, com total transparência de informações e decisões distribuídas. |

| *Possíveis fontes de evidências:* |
| :---- |
| **A) Artefatos e onde buscar** |
| Calendário de rituais interfuncionais (cadência, pauta e presença). |
| Atas de reuniões com lista de participantes e decisões. |
| Plataformas de comunicação (Slack, Teams, Confluence) com histórico de canais e threads. |
| RAACIs/matrizes de responsabilidade compartilhadas. |
| Portais de conhecimento e wikis corporativos com métricas de acesso. |
| Documentos de governança de comunicação e protocolos de compartilhamento de informação. |
| Acordos de colaboração e parcerias com stakeholders externos. |
| **B) Métricas/KPIs** |
| Taxa de participação em canais de comunicação por função/área. |
| Número de interações entre áreas (mensagens, reuniões, documentos compartilhados). |
| Tempo de resposta médio entre áreas. |
| Taxa de retrabalho ou falhas de comunicação. |
| Número de projetos/iniciativas interáreas concluídos por período. |
| Índice de transparência (% de decisões e informações documentadas e acessíveis). |
| NPS interno de comunicação e colaboração. |
| **C) Sinais por nível** |
| **N0:** Comunicação informal e verbal; silos organizacionais; ausência de documentação; handoffs não estruturados. |
| **N1:** Canais formais existem mas pouco utilizados; comunicação predominantemente top-down; informações concentradas em poucos indivíduos. |
| **N2:** Procedimentos de comunicação definidos; compartilhamento reativo; colaboração pontual mediada pela liderança; baixa autonomia das equipes. |
| **N3:** Reuniões regulares interfuncionais; equipes têm autonomia para ajustar processos de comunicação; início de compartilhamento proativo de informações. |
| **N4:** Comunicação bidirecional e transparente; dados quantitativos suportam discussões; responsabilidades e riscos parcialmente compartilhados; rotinas de comunicação bem estabelecidas. |
| **N5:** Canais integrados e transparentes; equipes multifuncionais se formam dinamicamente; metas e KPIs compartilhados; alta colaboração baseada em dados preditivos. |
| **N6:** Rede adaptativa com comunicação em tempo real; transparência total; decisões distribuídas; ajustes dinâmicos e autônomos; cultura de compartilhamento arraigada. |
| **D) Amostragem** |
| Selecionar 3-5 iniciativas recentes que exigiram colaboração entre áreas ou com parceiros externos e mapear o fluxo de comunicação (frequência, canais, participantes, tempo de resposta, qualidade das informações compartilhadas). |
| Entrevistar representantes de diferentes áreas sobre a efetividade dos canais de comunicação. |
| Analisar atas de reuniões interfuncionais dos últimos 90 dias. |

# **Questão 2 - Transparência organizacional**

| Como a organização compartilha informações estratégicas, decisões e mudanças com todos os níveis hierárquicos? |  |
| :---- | :---- |
| ***Capacidade em medição:*** | Comunicação aberta |

| *Respostas* |  |
| :---: | :---: |
| ***Peso*** | ***Item de resposta*** |
| 0 | Informações estratégicas são mantidas apenas na alta liderança, sem comunicação formal. Decisões não são compartilhadas ou justificadas. |
| 1 | Comunicação top-down básica apenas quando estritamente necessário. Informações compartilhadas de forma seletiva e tardia. |
| 2 | Informações são compartilhadas formalmente através de canais oficiais, mas de forma seletiva e com atraso significativo. Pouca justificativa das decisões. |
| 3 | Comunicação regular de informações estratégicas através de múltiplos canais formais. Decisões importantes são comunicadas com contexto básico. |
| 4 | Transparência estruturada com dashboards acessíveis, comunicação frequente e bidirecional. Decisões são explicadas com dados e contexto. |
| 5 | Transparência total com acesso em tempo real a informações estratégicas, métricas de desempenho e processos decisórios. Comunicação proativa e bidirectional. |
| 6 | Cultura de transparência radical com governança distribuída, comunicação contínua em tempo real e acesso universal a todas as informações relevantes. |

| *Possíveis fontes de evidências:* |
| :---- |
| **A) Artefatos e onde buscar**
Política de transparência e comunicação interna documentada.
Dashboards estratégicos com níveis de acesso por função/nível.
Atas de reuniões estratégicas (Town Halls, All-Hands) com registro de participação.
Canais de comunicação executiva (newsletters, vídeos, intranets).
Registros de decisões estratégicas com justificativas e contexto.
Portais de governança com visibilidade de projetos e iniciativas estratégicas.
Plataformas de perguntas e respostas (Q&A sessions, AMA - Ask Me Anything).
Sistema de gestão de mudanças com comunicação associada.
**B) Métricas/KPIs**
% de colaboradores que conhecem objetivos estratégicos da organização (pesquisa).
Frequência de comunicações executivas (mensal, semanal, diária).
Taxa de acesso a dashboards e portais de informação estratégica.
Tempo médio entre decisão estratégica e comunicação para toda organização.
Índice de clareza de comunicação (pesquisa de percepção).
Número de canais bidirecionais de comunicação ativos.
Taxa de engajamento em comunicações executivas (views, leituras, participação).
Índice de confiança na liderança (pesquisa de clima).
**C) Sinais por nível**
**N0:** Informações estratégicas totalmente restritas; colaboradores desconhecem direção organizacional; cultura de sigilo.
**N1:** Comunicação esporádica e reativa; informações liberadas apenas sob pressão; pouca visibilidade de decisões.
**N2:** Comunicação formal mas limitada; informações compartilhadas com atraso; pouca abertura para perguntas.
**N3:** Comunicação regular através de múltiplos canais; informações estratégicas acessíveis; decisões comunicadas com contexto básico.
**N4:** Transparência estruturada; dashboards acessíveis; comunicação frequente e bidirecional; decisões explicadas com dados.
**N5:** Transparência total; acesso em tempo real; comunicação proativa; alta responsividade a perguntas.
**N6:** Transparência radical; governança aberta; comunicação contínua; acesso universal; cultura de abertura total.
**D) Amostragem**
Analisar últimas 10 decisões estratégicas: tempo até comunicação, clareza da mensagem, canais utilizados, alcance.
Entrevistar 15-20 colaboradores de diferentes níveis sobre percepção de transparência.
Verificar acessibilidade de informações estratégicas em diferentes níveis hierárquicos.
Analisar engajamento em últimas 5 comunicações executivas.

# **Questão 3 - Cultura de feedback contínuo**

| Como a organização pratica e incentiva o feedback contínuo entre líderes, equipes e colaboradores? |  |
| :---- | :---- |
| ***Capacidade em medição:*** | Comunicação aberta |

| *Respostas* |  |
| :---: | :---: |
| ***Peso*** | ***Item de resposta*** |
| 0 | Não existe prática formal de feedback. Feedback acontece raramente e de forma ad hoc, geralmente apenas negativo ou em situações de crise. |
| 1 | Feedback anual via avaliação de desempenho formal. Processo burocrático e unidirecional (líder → colaborador). |
| 2 | Feedback trimestral ou semestral estruturado, mas ainda limitado ao fluxo top-down. Pouca abertura para feedback upward ou lateral. |
| 3 | Feedback mensal com canais formais estabelecidos. Início de feedback bidirecional (upward e downward), mas ainda irregular. |
| 4 | Feedback contínuo e multidirecional (upward, downward, lateral) com ferramentas digitais. Cultura que valoriza feedback construtivo. |
| 5 | Cultura forte de feedback em tempo real com feedback 360°, ferramentas automatizadas e alta segurança psicológica. |
| 6 | Feedback contínuo totalmente integrado ao fluxo de trabalho, com IA para análise de padrões, reconhecimento automatizado e coaching contínuo. |

| *Possíveis fontes de evidências:* |
| :---- |
| **A) Artefatos e onde buscar**
Política e processos de feedback documentados.
Registros de sessões de feedback (1:1s, check-ins, conversas de desenvolvimento).
Plataformas de feedback (Lattice, Culture Amp, 15Five, Betterworks).
Programas de feedback 360° com relatórios e planos de ação.
Guias e treinamentos sobre como dar e receber feedback.
Sistema de reconhecimento peer-to-peer.
Registros de PDI (Planos de Desenvolvimento Individual) vinculados a feedback.
Atas de retrospectivas e sessões de lessons learned.
**B) Métricas/KPIs**
Frequência de feedback formal e informal por colaborador.
% de colaboradores que receberam feedback no último mês/trimestre.
Taxa de participação em programas de feedback 360°.
Índice de qualidade do feedback (pesquisa de percepção).
Tempo médio entre feedback e plano de ação/melhoria.
% de líderes treinados em técnicas de feedback efetivo.
Taxa de utilização de ferramentas de feedback.
Correlação entre frequência de feedback e engajamento/performance.
**C) Sinais por nível**
**N0:** Ausência de feedback estruturado; feedback esporádico apenas em crises; cultura punitiva.
**N1:** Feedback anual burocrático; processo unidirecional; foco em avaliação passada sem desenvolvimento.
**N2:** Feedback periódico mas ainda top-down; pouca abertura para feedback lateral ou upward.
**N3:** Feedback mensal estabelecido; início de bidirecionalidade; canais formais existem.
**N4:** Feedback contínuo e multidirecional; ferramentas digitais facilitam; cultura valoriza feedback construtivo.
**N5:** Feedback em tempo real; 360° implementado; alta segurança psicológica; feedback integrado à cultura.
**N6:** Feedback contínuo no fluxo de trabalho; IA analisa padrões; coaching contínuo; feedback como hábito natural.
**D) Amostragem**
Analisar frequência e qualidade de feedback recebido por amostra de 20-30 colaboradores nos últimos 90 dias.
Verificar uso de ferramentas de feedback e taxa de engajamento.
Entrevistar colaboradores sobre percepção de qualidade e frequência de feedback.
Analisar correlação entre feedback recebido e evolução de performance.

# **Questão 4 - Comunicação em crises e mudanças**

| Como a organização comunica durante crises, mudanças significativas ou situações de incerteza? |  |
| :---- | :---- |
| ***Capacidade em medição:*** | Comunicação aberta |

| *Respostas* |  |
| :---: | :---: |
| ***Peso*** | ***Item de resposta*** |
| 0 | Silêncio ou comunicação muito inconsistente durante crises. Rumores e desinformação proliferam. Alta ansiedade organizacional. |
| 1 | Comunicação mínima e tardia, apenas quando pressionado. Mensagens vagas e pouco esclarecedoras. |
| 2 | Comunicação formal mas limitada e altamente controlada. Informações parciais, causando especulação e ansiedade. |
| 3 | Comunicação regular durante crises mas ainda predominantemente top-down. Atualizações periódicas com informações relevantes. |
| 4 | Comunicação transparente e frequente com espaço para dúvidas e preocupações. Liderança visível e acessível. |
| 5 | Comunicação em tempo real através de múltiplos canais bidirecionais. Alta transparência, empatia e responsividade a questões. |
| 6 | Comunicação radical e contínua com transparência total, canais bidirecionais ativos, co-criação de soluções e governança participativa. |

| *Possíveis fontes de evidências:* |
| :---- |
| **A) Artefatos e onde buscar**
Plano de comunicação de crises documentado.
Registros de comunicações durante crises/mudanças recentes (cronologia, canais, mensagens).
Atas de reuniões de gestão de crise.
FAQs (Frequently Asked Questions) e suas atualizações durante eventos críticos.
Transcrições de Town Halls e sessões de Q&A durante períodos de mudança.
Pesquisas de pulso (pulse surveys) durante momentos críticos.
Sistema de gestão de mudanças com planos de comunicação associados.
Canais de comunicação de emergência (hotlines, chats dedicados).
**B) Métricas/KPIs**
Tempo de resposta entre evento crítico e primeira comunicação oficial.
Frequência de comunicações durante período de crise/mudança.
Taxa de engajamento em comunicações de crise (views, leituras, participação em Q&A).
Índice de clareza e utilidade das comunicações (pesquisa).
Número de rumores/desinformações identificados e corrigidos.
Taxa de ansiedade/estresse organizacional durante crises (pulse surveys).
NPS de comunicação de crise (avaliação pós-evento).
Tempo médio de resposta a perguntas durante crises.
**C) Sinais por nível**
**N0:** Silêncio prolongado; rumores proliferam; alta ansiedade; desinformação; liderança invisível.
**N1:** Comunicação tardia e mínima; mensagens vagas; pouca clareza; liderança pouco acessível.
**N2:** Comunicação formal mas controlada; informações parciais; transparência limitada; pouco espaço para dúvidas.
**N3:** Comunicação regular top-down; atualizações periódicas; informações relevantes compartilhadas.
**N4:** Comunicação frequente e transparente; espaço para dúvidas; liderança visível e empática.
**N5:** Comunicação em tempo real; múltiplos canais; alta transparência; responsividade rápida a questões.
**N6:** Comunicação contínua; transparência radical; co-criação de soluções; governança participativa; confiança alta.
**D) Amostragem**
Analisar 2-3 crises ou mudanças significativas recentes: linha do tempo de comunicação, canais utilizados, conteúdo, frequência.
Realizar focus groups pós-crise para avaliar efetividade da comunicação.
Analisar dados de pulse surveys durante e após eventos críticos.
Comparar nível de ansiedade/confiança antes, durante e depois da crise.

# **Questão 5 - Escuta ativa organizacional**

| Como a organização pratica a escuta ativa, capturando e respondendo às preocupações, ideias e sentimentos dos colaboradores? |  |
| :---- | :---- |
| ***Capacidade em medição:*** | Comunicação aberta |

| *Respostas* |  |
| :---: | :---: |
| ***Peso*** | ***Item de resposta*** |
| 0 | Sem mecanismos de escuta. Comunicação totalmente unidirecional (top-down). Colaboradores não têm voz. |
| 1 | Mecanismos básicos de escuta (caixa de sugestões física/digital) existem mas são raramente utilizados e sem resposta. |
| 2 | Canais de escuta existem mas respostas são lentas, genéricas ou inexistentes. Colaboradores sentem que não são ouvidos. |
| 3 | Escuta estruturada com processos de captura e resposta estabelecidos. Fechamento do loop de feedback acontece mas ainda lento. |
| 4 | Escuta ativa com análise sistemática de inputs, respostas tempestivas e fechamento consistente do loop de feedback. |
| 5 | Plataformas de escuta multicanal com análise de sentimento, categorização automática e resposta ágil. Alta responsividade. |
| 6 | Escuta contínua em tempo real com IA, análise preditiva de sentimentos, identificação proativa de problemas e ação preventiva. |

| *Possíveis fontes de evidências:* |
| :---- |
| **A) Artefatos e onde buscar**
Política de escuta ativa e canais de voz dos colaboradores.
Plataformas de escuta (Glint, Peakon, Culture Amp, Officevibe).
Registros de sugestões, reclamações e ideias capturadas.
Sistema de gestão de feedbacks com status de cada input (aberto, em análise, resolvido, arquivado).
Atas de comitês que analisam inputs de colaboradores.
Relatórios de análise de sentimento e temas recorrentes.
Comunicações sobre ações tomadas baseadas em feedback dos colaboradores.
Pesquisas de clima, engajamento e pulse surveys com frequência e taxa de resposta.
**B) Métricas/KPIs**
Número de canais de escuta disponíveis e ativos.
Taxa de participação em pesquisas e canais de voz.
Tempo médio de resposta a inputs de colaboradores.
% de inputs que recebem resposta/ação.
Taxa de fechamento do loop de feedback ("você disse, nós fizemos").
Evolução do índice de sentimento organizacional ao longo do tempo.
% de colaboradores que sentem que suas vozes são ouvidas (pesquisa).
Número de melhorias implementadas baseadas em feedback de colaboradores.
**C) Sinais por nível**
**N0:** Ausência total de mecanismos de escuta; colaboradores não têm voz; cultura autoritária.
**N1:** Mecanismos simbólicos existem mas não funcionam; ausência de resposta; colaboradores desacreditam.
**N2:** Canais existem mas respostas são raras, tardias ou genéricas; percepção de não serem ouvidos.
**N3:** Escuta estruturada; processos estabelecidos; fechamento de loop acontece mas ainda lento.
**N4:** Escuta ativa; análise sistemática; respostas tempestivas; loop de feedback consistente.
**N5:** Escuta multicanal; análise de sentimento; categorização automática; alta responsividade.
**N6:** Escuta contínua em tempo real; IA preditiva; identificação proativa de problemas; ação preventiva.
**D) Amostragem**
Rastrear jornada de 20-30 inputs de colaboradores desde captura até resolução/resposta.
Analisar tempo de resposta e qualidade das respostas.
Entrevistar colaboradores sobre percepção de serem ouvidos.
Verificar taxa de utilização de canais de voz e satisfação com respostas.
Analisar % de inputs que geraram ação concreta.

| *Glossário* |
| ----- |
| **All-Hands Meeting:** Reunião com todos os colaboradores da organização onde liderança compartilha informações estratégicas e responde perguntas. |
| **AMA (Ask Me Anything):** Sessão aberta onde colaboradores podem fazer qualquer pergunta à liderança. |
| **Análise de sentimento:** Uso de tecnologia (geralmente IA) para identificar e categorizar opiniões e emoções expressas em texto. |
| **Canais bidirecionais:** Meios de comunicação que permitem fluxo de informação em ambas as direções. |
| **Canais formais de comunicação:** Meios estruturados e oficialmente estabelecidos pela organização para troca de informações (e-mail corporativo, plataformas de colaboração, reuniões regulares, sistemas de gestão). |
| **Colaboração inter e intraempresarial:** Trabalho conjunto dentro da organização (intra) e com parceiros externos (inter) para alcançar objetivos comuns. |
| **Comunicação aberta:** Prática organizacional que promove transparência, acessibilidade e bidirecionalidade no compartilhamento de informações. |
| **Comunicação bidirecional:** Fluxo de informações que ocorre em ambas as direções, permitindo feedback e diálogo entre partes. |
| **Cultura de feedback:** Ambiente organizacional onde feedback construtivo é esperado, valorizado e praticado regularmente em todas as direções. |
| **Dashboard:** Painel visual que apresenta métricas e indicadores-chave de desempenho de forma consolidada e acessível. |
| **Equipes multifuncionais:** Grupos compostos por membros de diferentes áreas ou especialidades trabalhando juntos em objetivos comuns. |
| **Escuta ativa:** Prática de ouvir atentamente, compreender, processar e responder de forma significativa aos inputs recebidos. |
| **Feedback 360°:** Processo de avaliação que coleta feedback de múltiplas fontes (superiores, pares, subordinados, clientes). |
| **Feedback upward:** Feedback que flui de subordinados para superiores na hierarquia organizacional. |
| **Fechamento do loop de feedback:** Prática de comunicar de volta aos colaboradores como seus inputs foram considerados e quais ações foram tomadas. |
| **Governança distribuída:** Modelo onde poder de decisão é compartilhado entre múltiplos atores em vez de centralizado. |
| **Handoff:** Processo de transferência de trabalho, informação ou responsabilidade entre pessoas, equipes ou áreas. |
| **KPI (Key Performance Indicator):** Indicador-chave de desempenho usado para medir o progresso em relação aos objetivos organizacionais. |
| **NPS interno:** Net Promoter Score aplicado internamente para medir a satisfação dos colaboradores com processos, ferramentas ou áreas da organização. |
| **PDI (Plano de Desenvolvimento Individual):** Documento que estabelece objetivos de desenvolvimento e aprendizado do colaborador. |
| **Pulse survey:** Pesquisa rápida e frequente (semanal/quinzenal) para capturar sentimento e engajamento em tempo quase real. |
| **Q&A (Questions and Answers):** Sessão estruturada de perguntas e respostas. |
| **RAACI/RACI:** Matriz que define papéis e responsabilidades (Responsible, Accountable, Consulted, Informed, ou variações). |
| **Rede adaptativa:** Estrutura organizacional flexível que se reconfigura dinamicamente conforme as necessidades e contextos mudam. |
| **Segurança psicológica:** Crença compartilhada de que o ambiente é seguro para assumir riscos interpessoais, como expressar opiniões divergentes ou admitir erros. |
| **Silos organizacionais:** Situação em que áreas ou departamentos operam isoladamente, com pouca comunicação ou colaboração entre si. |
| **Stakeholders externos:** Partes interessadas fora da organização (fornecedores, clientes, parceiros, instituições). |
| **Tempo real:** Informação ou comunicação que acontece instantaneamente ou com atraso mínimo. |
| **Town Hall:** Grande reunião organizacional onde liderança comunica informações importantes e responde perguntas. |
| **Transparência organizacional:** Prática de tornar informações, decisões e processos visíveis e acessíveis aos membros relevantes da organização. |
| **Transparência radical:** Nível extremo de abertura onde praticamente todas as informações são compartilhadas com todos os colaboradores. |
